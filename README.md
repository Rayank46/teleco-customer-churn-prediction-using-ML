# teleco-customer-churn-prediction-using-ML
This project focuses on analyzing customer behavior and predicting customer churn using machine learning. The system processes customer data, performs exploratory data analysis (EDA), creates useful behavioral features, and identifies patterns that influence customer retention. Machine learning models such as Logistic Regression and Random Forest are used to predict churn probability, while customer segmentation helps classify customers into different value groups. The final solution includes visualizations, business insights, churn risk analysis, and an interactive Streamlit dashboard for easy exploration of results and predictions.
